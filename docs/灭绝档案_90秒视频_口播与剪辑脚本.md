# 《灭绝档案：失落物种的人工智能纪念馆》

**90 秒感性向 · 口播稿 + 剪辑分镜脚本**

> 用途：口播录制、剪辑分轨、字幕与素材清单对齐。  
> 目标时长：**85–95 秒**（可按语速 ±5 秒微调）。  
> 画幅默认：**16:9**（竖屏 9:16 可保留时间段，改构图即可）。

## 二、声音总规（给混音）

- **音乐**：低频铺底 + 轻弦乐/氛围电子；**避免大鼓点**。问题段更空，结尾最后 3 秒可微抬一次。
- **旁白 VO**：干净、句间留 **0.2–0.4s** 呼吸。
- **环境音（极轻，可选）**：风/海浪/远处鸟叫，像回忆层，不要写实抢戏。
- **UI 音（可选）**：点光点一次极短「叮」（约 0.1s），其余不要。

## 三、完整口播稿（约 90 秒 · 感性版）

> 本稿为**展览解说体**：避免「我、你」等第一、二人称，便于公开展映与多语种配音。可直接整段录制，再由剪辑按分镜切开。

**【开场】**  
有一种安静，是博物馆里的骨骼；还有一种安静，是地图上再也点不到的坐标。人们常说「它灭绝了」，可这句话落下之后，真正消失的是什么？是名字，是身影，还是人与它们曾共处过的那一段时间？

**【立场】**  
这里关心的并不是让过去「复活」，而是承认：许多消失并不是遥远的自然法则，而与人类曾走过的路、开过的地、要过的价有关。叙述尝试把每一次离开，从一句结论，拉回到它曾在何处生活、以何种方式被推向终点。

**【项目】**  
于是出现《灭绝档案：失落物种的人工智能纪念馆》这一面向公众的入口——以一颗会转的地球，将物种重新安放进地理与时间：每一枚光点，都曾对应有体温的生命；每一次点选，都是从「知道它已不在」到「理解它如何离去」的一次靠近。

**【调研】**  
调研中，故事被尽量拦在传说与口号之外。红色名录中的句子、旧文献里的描摹、馆藏编号、分布记录中残存的一线痕迹——这里追求的不是更炫的技术，而是一份经得住反复对照的集体记忆：对叙述可以持保留态度，但始终留得出可回去核对的路标。

**【设计】**  
页面上，「档案」被放轻，「抵达」被加重：先看见世界，再看见个体。当视线跟随地球缓慢旋转，会有一种很朴素的感受——那重量已难以被握在掌心，却仍能在叙述与记忆中被试着安放回原位。

**【AI】**  
人工智能在这里并不扮演造物主。它更像一位谨慎的编目员：把散落的材料拼成可读的句群，让图像与声音、路径与问题并置——在不确定中仍保留可讲述之物。

**【收束】**  
对最初问题的一种回答或许是：物种灭绝最残酷之处，并不在于它发生了一次，而在于它发生之后，人类是否仍能重新学会在场。

**【片尾字卡（可单独录静音层或出静帧）**  
为失落物种，保留一种被理解的方式。  
——《灭绝档案》

## 六、三条剪辑红线（给剪辑师）

1. **前 12 秒**少堆 UI 字，先情绪与问题，再进项目与操作。  
2. 操作段只出现 **手 / 光标 / 光点 / 地球** 四类画面，大段技术 dock 能剪则剪。  
3. 项目**长标题**在片中建议最多 **2 次**清晰出现：项目名片段 1 次 + 尾板 1 次，避免碎。
