# Animal Migration Routes

This file documents the route data used by the WebXR migration demo. The prototype species are extinct or data-limited, so routes are marked `prototype_inferred_from_historical_range`: they are visual storytelling paths inferred from historical region/category context, not GPS tracks. Island species use shorter habitat-shift paths. Replace coordinates with peer-reviewed or curator-provided routes when available.
