核心概念（30 秒版）

问题：物种灭绝带走的不仅是 DNA，还有其在昼夜与季节中的时间生态位；公众记忆常被压成图像，而非「在世界中的时间」。

方案：WebXR 纪念空间，重建猛犸象与袋狼等案例的节律与感官线索；界面区分 Cited / Interpolated / Speculative，并设伦理分叉与反思环节。
