How to open (Cursor / browser)

  • Browser: In Finder / Explorer, open this file: slides/preview.html (double-click or drag into Chrome / Edge / Safari). Relative links to PDF/PPTX work from this page.
  • Cursor: Right-click preview.html → Open with… your browser, or install the Live Preview extension and use “Show Preview” (some setups need a workspace root so links resolve).
  • Edit slides: Install Marp for VS Code, open the .md deck, use the Marp preview pane while editing.

Note: There is no automatic public URL for files on your laptop. To share online, upload export/*.pdf to Drive / Dropbox / GitHub Releases and replace links accordingly.
