Slide 02

The fracture

  • Extinction removes species and ecological function
  • It also erodes shared memory: shifting baselines, hype cycles (de-extinction as spectacle)
  • We lose temporal coexistence: migrations, synchrony, daily and seasonal rhythms

This project is not “resurrection.” It makes absence and evidence discussable.

Slide 03

Design questions

  • How do we translate extinction–absence into perceivable, debatable experience?
  • How can AI + biological data support disciplined public memory—without faking nature?
  • How do we land in accountability (conservation tradeoffs), not only immersion?

Slide 07

Bio + Digital (both load-bearing)

Biology: paleo-sensory proxies, isotope trajectories, ethology, extinction mechanisms — traceable sources

Digital: conditional generative visuals (B) + procedural / shader base (A); Web Audio; WebAR + desktop fallback

Remove either layer and the project stops making sense.

Honesty UI: three tiers

  • Cited — direct literature / dataset

  • Speculative — creative extension; never sold as footage
