---

# Why this project now

Extinction removes not only bodies, but also ways of being in time:

- migration pulses
- collective synchrony
- circadian fit to landscape
- remembered abundance

Public memory often goes quiet after species disappear.  
This project asks how constrained computation can make absence legible without pretending we fully know another species' Umwelt.

---

# Design proposition

- **Primary:** Umwelt as a bounded hypothesis space
- **Secondary:** collective memory fracture as the ethical frame
- **Experience path:** Memory Sites → Species Dossier → Polyphony Mixer → Ethics Console

The archive is not resurrection.  
It is an evidence-scaffolded memorial that keeps uncertainty visible.

---

# Three load-bearing modules

1. **Species Dossier**  
   Structured evidence cards, sensory proxies, extinction mechanisms, and controlled immersive scenes.

2. **Polyphony Mixer**  
   Seasonal, diel, acoustic, and density layers become audible ecological structure.

3. **Ethics Console**  
   Visitors classify evidence, face tradeoffs, and hear consequences in the final mix.

# Biological layer

Only published or clearly framed modeled claims enter the archive:

- migration isotopes
- morphology as sensory proxy
- comparative genomics
- acoustic ecology of relatives
- habitat and vegetation references

Every strong line is tagged:

`Cited` · `Modeled` · `Speculative`

---

# Digital layer

- **Primary generation:** diffusion + structural conditioning
- **Support layer:** procedural shaders for light, fog, atmosphere, and credible scene logic
- **XR shell:** A-Frame-first web experience
- **Science visuals:** R / ggplot exports
- **Sound:** Web Audio DSP tied to season, density, and time-of-day

The digital layer is expressive, but never unconstrained.

---

# Physical and living contrast

- 3D-printed focal object
- mobile WebAR deep links
- tactile material board
- plant + soil moisture/light/temperature bench

The living bench is not a proxy for extinct habitat.  
It indexes what is still alive now, and what remains under present ecological care.

---

# Ethics and reflection

- de-extinction hype is kept separate from conservation urgency
- AI speculation is visibly labeled
- resource allocation decisions are not all winnable
- land-use and justice remain present without ventriloquizing communities

Reflection is not an appendix.  
It is part of the interface.
